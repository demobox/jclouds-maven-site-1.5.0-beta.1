66/**

68 * <p/>
69 * Particularly, this code delegates calls to other things.
70 * <ol>
71 * <li>if the method has a {@link Provides} annotation, it responds via a
72 * {@link Injector} lookup</li>
73 * <li>if the method has a {@link Delegate} annotation, it responds with an
74 * instance of interface set in returnVal, adding the current JAXrs annotations
75 * to whatever are on that class.</li>

79 * </li>

83 * it responds with an http implementation.</li>
